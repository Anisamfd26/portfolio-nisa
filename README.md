# Portfolio Nisa - Digital Ecosystem & Business Insights

Welcome to my personal portfolio repository. This project is a digital ecosystem built to showcase my journey as a **Computer Science student** with a strong passion for **Data Analysis** and **Product Management**.

## 🚀 Vision
To build a digital-based ecosystem capable of solving real-life problems through data-driven innovation that is not only technically efficient but also delivers a significant growth impact.

## 🛠️ Tech Stack
This portfolio is built using modern web technologies:
* **Framework:** React.js with TypeScript
* **Styling:** Tailwind CSS & Lucide Icons
* **Runtime & Package Manager:** Bun (using `bun.lock`)
* **Testing:** Vitest & Playwright (for reliable UI/UX)
* **Components:** Radix UI / Shadcn UI (based on `components.json`)

## 📂 Project Structure
* `src/components/sections`: Contains modular sections like Experience, Skills, Projects, and Achievements.
* `src/data/portfolio.ts`: The central "source of truth" for all portfolio content, making it easy to manage data.
* `src/hooks`: Custom React hooks for theme management and mobile responsiveness.

## 💼 Business & Management Highlights
Beyond coding, this project reflects my experience in:
* **Product Ownership:** Structured information architecture (IA) similar to my work at PT Daltek Indonesia.
* **Project Management:** Organized workflow using Jira and Notion principles.
* **Data Driven:** Designed to present achievements and skills through a clean, analytical interface.

## ⚙️ How to Run Locally

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Anisamfd26/portfolio-nisa.git](https://github.com/Anisamfd26/portfolio-nisa.git)
    ```
2.  **Install dependencies (Recommended using Bun):**
    ```bash
    bun install
    ```
3.  **Run the development server:**
    ```bash
    bun dev
    ```
4.  **Build for production:**
    ```bash
    bun run build
    ```
